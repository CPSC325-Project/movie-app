Getting started:  
1. clone
1. run `npm install`
1. run `npm install firebase`
1. if you get a module error not found when running locally --> run `npm ci`


To run locally:
1. run `npm run dev`

To redploy:
1. run `npm run build`
1. run `firebase deploy`
**if you don't see your changes then try clearing your cookies & cache then redploy**
